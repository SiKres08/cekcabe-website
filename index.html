apa itu
cek cabe?

Aplikasi identifikasi penyakit tanaman cabai berbasis AI dengan model Convolutional Neural Network yang dapat mengidentifikasi penyakit tanaman cabai secara langsung hanya dengan gambar.

kenapa cek cabe?

Identifikasi Cepat

Identifikasi penyakit berlangsung cepat karena model Edge-AI

Akurasi Tinggi

Karena model dilatih dengan ribuan gambar dataset

Offline Mode

Identifikasi tetap berjalan tanpa internet

Ramah Pengguna

Desain UI sederhana dan mudah digunakan

3 Langkah Cek Cabe

Hanya terdapat 3 langkah simple dalam proses identifikasi tanaman cabai

01

Pilih Input

Dapat melalui ambil foto dengan kamera aplikasi atau unggah gambar tanaman cabai dari galeri

02

Input Gambar

Dengan menekan tombol shutter kamera aplikasi atau memilih gambar tanaman cabai di galeri internal smartphone

03

Lihat Hasil

Tampilan hasil identifikasi penyakit tanaman cabai, beserta informasi mengenai penyakit hasil identifikasi
